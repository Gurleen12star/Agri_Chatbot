🌾 AgriChatbot – Smart Farming Assistant

(Tasks 1 → 4 Integrated Project)

An AI-powered agriculture chatbot that helps farmers with queries, crop insights, multilingual support, and image-based crop analysis.

✨ Features
💬 Intelligent chatbot for agriculture-related queries
🔐 User authentication (Register/Login system)
👨‍💼 Admin dashboard for monitoring users and activity
🌐 Multilingual support (English, Hindi, Punjabi)
📷 AI-powered crop image analysis
🤖 OpenAI integration for smart responses
📊 Scalable architecture with modular task-based development

⚙️ How It Works
User logs in or registers
User interacts with chatbot
Query is processed using OpenAI API
Response is returned in selected language
(Optional) User uploads crop image
AI analyzes image and provides insights

⚙️ Installation & Setup
# Clone repository
git clone https://github.com/yourusername/agrichatbot.git

# Navigate to task folder
cd task4   # or task1 / task2 / task3

# Install dependencies
pip install -r requirements.txt

# Run application
python app.py

App will run on:
👉 http://127.0.0.1:5000
